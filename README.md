# Infinity XOS 2.0

**Canonical Monorepo** — Production-ready AI Operating System

## System Truth

- **Public Domain**: https://infinityxai.com
- **Control Plane**: https://infinityxai.com/admin
- **Stack**: Google Cloud (Cloud Run + Firestore + Pub/Sub + Secret Manager + GCS + Vertex AI)
- **Local Dev**: Docker + VS Code + MCP bridge + Ollama

## Repository Structure

```
/apps
  /web                 # infinityxai.com frontend (React + Tailwind)
  /api                 # Cloud Run API gateway (Node.js + Express)
/packages
  /contracts           # versioned OpenAPI + contract hash tooling
  /typed-client        # generated client from OpenAPI
  /shared              # shared types/utils
/ops
  /scripts             # hydrate, preflight, deploy-staging, deploy-prod, rollback, verify, run-demo
  /runbooks            # preflight, launch, incident, backup/restore drill
  /reports             # evidence packs autogenerated
/docs/spec             # v2 specifications (159 canonical docs)
/.github/workflows     # CI, security scan, deploy staging/prod, contract hash gate
```

## Development

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Deploy staging
npm run deploy:staging

# Deploy production
npm run deploy:prod
```

## Contract-First Development

All API changes must:
1. Update `/packages/contracts/openapi.yaml`
2. Generate typed client via CI
3. Pass contract hash verification

## Security

- **No secrets in code** — Secret Manager only
- **PR-based workflow** — No direct pushes to main
- **Least privilege IAM** — Documented roles
- **Audit logs** — All admin actions tracked
